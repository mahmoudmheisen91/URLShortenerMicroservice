URL Shortener Microservice is a RESTful API that allow the client to shorten a URL, the server will direct to the original link, done for FreeCodeCamp APIs and Microservices Course, Project 3


Usage:

POST URL: POST /api/shorturl/new {"original_url": "www.google.com", "short_url": 1}

GET All URLs: GET /api/shorturl/urls

GET Short URL: GET /api/shorturl/[&index]

[ ] = required
